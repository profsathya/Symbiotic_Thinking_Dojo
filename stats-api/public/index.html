Symbiotic Thinking Dojo - Stats API

This API provides anonymous, aggregate usage statistics for the Symbiotic Thinking Dojo.

Endpoints

GET /.netlify/functions/stats

Returns aggregate statistics for the last 30 days (or specify ?days=N).

View current stats

POST /.netlify/functions/track

Records anonymous usage events. Used internally by the Dojo app.

Data Transparency

All statistics are anonymous and aggregate. We track:

  • Session counts by date
  • Conversation length distribution
  • Average DIKW levels
  • Sparring partner popularity
  • Practice Dojo topic usage

No personally identifiable information is collected.
